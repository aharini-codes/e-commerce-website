# 🛒 E-Commerce Website

A responsive **E-Commerce Website** developed using **HTML, CSS, Bootstrap, and JavaScript**.  
This project simulates a simple online shopping interface where users can browse products, search for items, and add them to the cart.

The website demonstrates front-end development skills including responsive design, UI components, and basic JavaScript functionality.

---

## 🚀 Features

- Product listing with images
- Product search functionality
- Add to Cart button with cart counter
- Responsive layout using Bootstrap
- Product hover animation
- Hero banner section
- Footer section
- Clean and user-friendly interface

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
